## Summary

| Component | Description | Connection |
|-----------|-------------|------------|
| **Pages** | UI Screens | Uses BLoC for state |
| **BLoC** | Business Logic | Uses Repository for data |
| **Repository** | Data Access | Uses DataSource for API |
| **DataSource** | API Calls | Uses ApiClient for HTTP |
| **ApiClient** | HTTP Client | Calls Backend API |
| **Backend** | FastAPI Server | Returns JSON Response |
| **State** | Data Model | Updated by BLoC |
| **Event** | User Action | Triggers BLoC |
